Tema Aplikasi: ZaloraLite (aplikasi sederhana dan ringan untuk digunakan)
Aplikasi rujukan: Zalora
Fitur yang dibuat: Daftar, Login, Menu Home, Menu Tas, Menu Wishlist, Menu Akun Saya

Cara menjalankan aplikasi:
1. Buat akun terlebih dahulu
2. Setelah itu Login
3. Memilih salah satu barang/produk yang kita inginkan
4. Klik "Tambahkan ke Tas" jika ingin membelinya, maka barang tersebut akan masuk ke menu Tas
5. Masukkan ke wishlist jika ingin menyimpan produk yang disukai atau ingin dibeli di kemudian hari, maka barangnya akan masuk ke menu Wishlist
6. Jika ingin men-checkout barang yang di inginkan, masuk ke dalam menu Tas lalu tekan tombol "Checkout".
